# Real-time-Bayesian-reconstruction-of-magnetic-equilibria-and-kinetic-profiles-for-the-WEST-tokamak
The ambitious aim of this repository is to provide a comprehensive and fully detailed diagram of the complete Bayesian model used in this project. Complete mathematical rigour of the whole model, together with the specific values of each hyperparameter used in the model employed.

This work corresponds to the master's thesis of Iván Alcaraz Aguado (https://orcid.org/0009-0009-9481-3494) for the Fusion-EP Master's Programme (European Master of Science in Nuclear Fusion and Engineering Physics).
